Module 02

เริ่มต้นใช้ Claude Cowork

การเริ่มใช้งานและ workflow พื้นฐาน

2.1 สมัครและติดตั้ง Claude Cowork

ข้อกำหนด: ต้องสมัครแผน Pro ขึ้นไป ($20/เดือน)

ขั้นตอนทั้งหมด:

  1. สมัคร Claude Account → claude.ai → Sign Up
  2. อัปเกรดเป็น Pro → claude.ai/upgrade
  3. ติดตั้ง Claude Desktop (Windows/Mac) → claude.com/download → ติดตั้ง → เปิดแอป → Login
  4. ตั้งค่า Privacy → ปิดการนำข้อมูลไปเทรนต่อ (สำคัญ! ข้อมูลบริษัทต้องเป็นความลับ)
  5. แนะนำตัวให้ AI รู้จักเรา → ใช้ Prompt ด้านล่าง
✦ Prompt
ฉันเป็นพนักงาน [ชื่อฝ่าย] ของบริษัท มารีนโกลด์โปรดักส์ จำกัด (MRG) ธุรกิจหลักคือการผลิตและส่งออกผลิตภัณฑ์กุ้งและอาหารทะเลแช่แข็งระดับโลก งานของฉันเกี่ยวกับ [อธิบายงานหลัก] ลูกค้าของเราอยู่ใน [ประเทศ/ภูมิภาค] ภาษาที่ใช้ในงาน: ภาษาไทย และ ภาษาอังกฤษ โปรดช่วยสรุปโปรไฟล์ของฉันเป็น System Prompt สั้นๆ ไม่เกิน 1,500 ตัวอักษร เพื่อให้คุณจำและทำงานร่วมกับฉันได้ดีขึ้น
💡Tip: นำข้อความที่ได้ไปวางในส่วน Memory หรือ User Preferences ของ Claude → AI จะตอบได้ตรงบริบทงาน MRG ทุกครั้ง

2.3 Prompt แรก — ลองสั่งงาน AI

✦ Prompt
สร้างไฟล์ Excel รายการสินค้าส่งออกกุ้ง มีคอลัมน์: ลำดับ, รหัสสินค้า, ชื่อสินค้า (ไทย), Product Name (ENG), น้ำหนักสุทธิ (kg), น้ำหนักรวม (kg), ราคา FOB (USD), ตลาดส่งออก ใส่ตัวอย่างข้อมูล 10 รายการ สำหรับกุ้งขาวแช่แข็ง จัดรูปแบบให้สวยงาม มีสีหัวตาราง

ขั้นตอน:

  1. Copy Prompt → Paste ใน Claude Cowork → กด Enter
  2. รอ AI ทำงาน (สร้างไฟล์ Excel ให้อัตโนมัติ)
  3. กดลิงก์ดาวน์โหลดไฟล์ .xlsx ที่ AI ส่งมา
💡Tip: ถ้าผลลัพธ์ยังไม่ตรงใจ พิมพ์ต่อได้ทันที เช่น "เพิ่มคอลัมน์ Lot Number" หรือ "แปลงราคาเป็น THB ด้วย"
